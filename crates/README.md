# KeyCortex Rust Crates

This folder contains shared Rust crates for wallet core logic, chain adapters, storage, crypto, auth adapter, and API contracts.

Current crates:

- `kc-api-types`
- `kc-crypto`
- `kc-wallet-core`
- `kc-storage`
- `kc-chain-client`
- `kc-chain-flowcortex`
- `kc-auth-adapter`
